Guía de OSINT
🔍 Ingresa un email para investigar...
📖 Guía rápida: OSINT (Open Source Intelligence) recopila información pública sobre un email. Detecta breaches, dominios asociados y da recomendaciones de seguridad.
Guía de Rainbow Tables
📝 Esperando acción...
📖 Guía rápida: Rainbow Tables permiten descifrar hashes usando diccionarios precomputados. Útil para recuperar contraseñas olvidadas o demostrar vulnerabilidades de hashes débiles.
Guía de WHOIS
🔍 Ingresa un dominio para consultar WHOIS...
📖 Guía rápida: WHOIS consulta información de registro de dominios: propietario, fechas de creación/expiración, servidores DNS. Esencial para investigaciones de dominio y OSINT.
Guía de Wappalyzer
🔍 Ingresa una URL para analizar tecnologías...
📖 Guía rápida: Wappalyzer detecta tecnologías web: frameworks, servidores, CMS, analytics. Útil para reconocimiento en pentesting y análisis de competencia.
Guía de Subdomain Finder
🔍 Ingresa un dominio para buscar subdominios...
📖 Guía rápida: Subdomain Finder descubre subdominios asociados a un dominio principal. Esencial para mapear superficies de ataque en auditorías de seguridad.
