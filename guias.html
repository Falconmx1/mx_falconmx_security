Guías de Herramientas

Aprende a usar cada herramienta de MFH TOOLS PRO con estas guías completas

🔍 Escáner de Puertos
TCP/UDP Nmap

El escáner de puertos permite detectar servicios activos en un objetivo. Ingresa una IP o dominio y un rango de puertos (ej: 22,80,443). Activa el modo Tor para escaneos anónimos. Los resultados se exportan a JSON para análisis posterior.

5 min Intermedio
📖 Ejemplo: Escanear scanme.nmap.org en puertos 22,80,443
🔐 RSA/AES Cipher
Cifrado Criptografía

RSA/AES permite cifrar y descifrar mensajes. AES es simétrico (misma clave para cifrar/descifrar). RSA es asimétrico (clave pública/privada). Úsalo para proteger información sensible o enseñar conceptos de criptografía.

3 min Básico
📖 Ejemplo: Cifrar "Mensaje secreto" con clave "MiClave123"
🕵️ OSINT Recon
Inteligencia Breaches

OSINT (Open Source Intelligence) recopila información pública sobre un email. Detecta breaches, dominios asociados y da recomendaciones de seguridad. Esencial para investigaciones de seguridad y protección de datos.

4 min Intermedio
📖 Ejemplo: Investigar ejemplo@dominio.com
🌈 Rainbow Tables
Hash MD5/SHA

Rainbow Tables permiten descifrar hashes usando diccionarios precomputados. Útil para recuperar contraseñas olvidadas o demostrar vulnerabilidades de hashes débiles. Soporta MD5, SHA1 y SHA256.

3 min Intermedio
📖 Ejemplo: Descifrar 5d41402abc4b2a76b9719d911017c592 → "hello"
🌐 WHOIS Lookup
Dominio DNS

WHOIS consulta información de registro de dominios: propietario, fechas de creación/expiración, servidores DNS. Esencial para investigaciones de dominio y OSINT. Ideal para auditors de seguridad y pentesters.

2 min Básico
📖 Ejemplo: Consultar google.com
📊 Wappalyzer
Tecnologías Web

Wappalyzer detecta tecnologías web: frameworks, servidores, CMS, analytics. Útil para reconocimiento en pentesting y análisis de competencia. Identifica React, Angular, Nginx, Apache y más.

3 min Intermedio
📖 Ejemplo: Analizar https://github.com
🗺️ Subdomain Finder
Subdominios Enumeración

Subdomain Finder descubre subdominios asociados a un dominio principal. Esencial para mapear superficies de ataque en auditorías de seguridad. Encuentra mail, api, dev, blog, shop y más.

4 min Avanzado
📖 Ejemplo: Buscar subdominios de dominio.com
🔢 Hash Tools
Generador Verificador

Hash Tools genera hashes MD5, SHA1, SHA256 y SHA512. También verifica la integridad de datos comparando hashes. Fundamental para verificar descargas y archivos, y para enseñar conceptos de hash.

2 min Básico
📖 Ejemplo: Calcular SHA256 de "Hola mundo"
🤖 IA Asistente
IA Logs

El Asistente IA clasifica logs de seguridad como maliciosos, sospechosos o benignos. Ideal para SOC, equipos de respuesta a incidentes y análisis forense. Detecta patrones de ataques y da recomendaciones.

5 min Avanzado
📖 Ejemplo: Analizar log de SSH con fallos de autenticación
🔑 Password Generator
Contraseñas Seguridad

Genera contraseñas seguras con longitud variable, mayúsculas, minúsculas, números y símbolos. Sigue las mejores prácticas de NIST para contraseñas fuertes. Ideal para crear credenciales seguras.

2 min Básico
📖 Ejemplo: Generar contraseña de 16 caracteres con todo incluido
📂 Hash Compare
Comparador Integridad

Compara dos archivos calculando sus hashes y verificando si son idénticos. Útil para verificar integridad de archivos descargados o detectar modificaciones. Soporta MD5, SHA1 y SHA256.

3 min Intermedio
📖 Ejemplo: Comparar dos versiones de un mismo archivo
🌍 HTTP Headers
Cabeceras Seguridad

Analiza cabeceras HTTP para identificar configuraciones de seguridad (HSTS, CSP, X-Frame-Options). Esencial para auditorías de seguridad web y pentesting. Detecta configuraciones débiles.

3 min Intermedio
📖 Ejemplo: Analizar cabeceras de https://google.com
🔗 URL Encode
Codificar URL

URL Encode convierte caracteres especiales a formato URL-safe (%20 para espacios). Útil para construir URLs, depurar parámetros y enseñar fundamentos web. Codifica y decodifica texto.

2 min Básico
📖 Ejemplo: Codificar "Hola mundo" → "Hola%20mundo"
💻 IP Converter
IP Redes

IP Converter muestra una IP en formato binario, hexadecimal y decimal. Útil para redes, subnetting, y comprender la estructura de direcciones IP. Esencial para estudiantes y profesionales de redes.

2 min Básico
📖 Ejemplo: Convertir 192.168.1.1 a binario y hexadecimal
🕐 Time Zones
Zonas Horarias Forense

Time Zones muestra la hora actual en múltiples zonas horarias (UTC, Local, UNIX timestamp). Esencial para análisis forense, correlación de eventos y coordinación internacional de equipos de seguridad.

2 min Básico
📖 Ejemplo: Ver hora UTC, Local y UNIX timestamp actuales
